WeCook is a platform for long horizontal robotic manipulation and human-robot collaboration research.

Setup Instructions

WeCook depends on AIKIDO, developed by the Personal Robotics Lab at the University of Washington.

WeCook is supported on the following systems:

  • Ubuntu 18.04 / ROS Melodic
  • Ubuntu 16.04 / ROS Kinetic

Demo

Fruit Jelly

To run the demo, type in following commands in your terminal

# Open one terminal
$ roscore
# Open another terminal
$ rviz
# Open another terminal
$ rosrun wecook wecook
# Open another terminal
$ rosrun wecook fruit_jelly.py
